# Azure Deployment (Stage 2)

This directory will contain Azure deployment configurations for:

## Planned Components

### Azure Container Registry + Container Apps
- Push Docker image to ACR
- Deploy FastAPI app via Azure Container Apps
- Auto-scaling configuration

### Azure Blob Storage
- Document upload storage
- Lifecycle management policies

### Azure Function
- Event-driven document processing
- Triggered on Blob Storage upload
- Automatic analysis pipeline execution

### Monitoring
- Application Insights integration
- Custom metrics for model inference latency
- Health check dashboards

## Prerequisites
- Azure CLI installed and configured
- Azure subscription with appropriate permissions
- Terraform (optional, for IaC)
